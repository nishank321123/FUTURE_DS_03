# FUTURE_DS_03
# 📊 College Event Feedback Analysis

### Data Science & Analytics Internship – Task 3

This project analyzes **student feedback collected from college events** to understand satisfaction levels, identify common issues, and generate insights that can help improve future events.

The analysis includes **data cleaning, exploratory data analysis (EDA), sentiment analysis, and visualizations** to transform raw feedback into meaningful insights.

---

# 🎯 Project Objective

The main objective of this project is to:

* Analyze student event feedback data
* Identify satisfaction trends among attendees
* Perform **sentiment analysis on feedback comments**
* Visualize patterns in ratings and responses
* Provide **actionable recommendations** for improving future events

---

# 🛠 Tools & Technologies Used

* **Python**
* **Google Colab / Jupyter Notebook**
* **Pandas** – Data manipulation
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical charts
* **TextBlob / VADER** – Sentiment analysis

# 📈 Key Analysis Performed

### 1️⃣ Data Cleaning

* Removed missing values
* Processed text feedback
* Prepared dataset for analysis

### 2️⃣ Exploratory Data Analysis

* Distribution of event ratings
* Frequency of responses
* Identification of common patterns

### 3️⃣ Sentiment Analysis

Feedback comments were analyzed using **NLP techniques** to classify them into:

* Positive
* Neutral
* Negative

### 4️⃣ Data Visualization

Various charts were created to understand trends such as:

* Rating distribution
* Sentiment breakdown
* Feedback insights

---

# 📊 Sample Insights

* Majority of students rated the event **4 or 5 stars**
* Most feedback was **positive**
* Negative feedback mainly focused on **event management and timing**
* Workshops received **higher satisfaction ratings**

---

# 💡 Recommendations

Based on the analysis:

* Improve **event organization and scheduling**
* Increase **interactive activities**
* Enhance **audio/visual arrangements**
* Collect structured feedback for better analysis in future events
